# TripPlan 企画書（フル版）

## 1. 企画概要
TripPlan は、旅行計画に必要な「旅程・持ち物・メモ」を一元管理する  
モバイルファーストな旅行計画 Web アプリケーションである。

### コンセプト
**Plan. Pack. Go.**  
計画 → 準備 → 出発までを、ひとつの体験としてシンプルにする。

---

## 2. 課題定義

### 見つけた課題
- 旅行情報が複数ツールに分散している
- グループ旅行で最新情報の共有が難しい
- スマートフォンでの確認にストレスがある

### 課題の本質
旅行計画・情報管理・共有を  
**モダンで直感的な UI で一元化できていないこと**

---

## 3. 目的・提供価値

### 解決する課題
旅行計画・準備・実行における煩雑さとストレス

### 提供価値
- 情報の一元管理
- モバイル前提の UI/UX
- 旅行体験そのものに集中できる時間

---

## 4. ターゲットユーザー
- 少人数グループ旅行の幹事
- 個人旅行者
- デジタルネイティブ層

---

## 5. 機能一覧

### MVP
- ユーザー登録 / ログイン
- マイ旅一覧
- 旅の作成・編集・削除
- 旅程（Activities）
- 持ち物リスト
- メモ

### 将来対応
- 公開リンク
- 並び替え
- 共同編集

---

## 6. 業務ルール（概要）

### Trip
- 開始日・終了日は必須
- start_date <= end_date
- 削除時は関連データも削除

### Activity
- 必須：title / date
- 任意：time / location / cost / memo
- 自動ソート

### Packing Item
- カテゴリ固定
- チェック機能あり

---

## 8. 非機能要件

---

## 9. まとめ
TripPlan は、実装・拡張を前提に  
MVPとして完成度の高い旅行計画アプリを目指す。
